# 📊 Madhav Ecommerce Sales Dashboard

## 🔍 Overview
This project analyzes ecommerce sales data using Power BI.

## 📈 Key Insights
- Maharashtra has highest sales
- Clothing category dominates (63%)
- COD is most used payment method (44%)

## 🛠 Tools Used
- Power BI
- Excel
